What we do

Meet. Hack.
Build together.

Non-competitive, open, community-driven hackathons — rotating between cities across the Asia-Pacific.

🤝

Connect

We bring together civic tech communities — Code for Japan, Code for Korea, g0v (Taiwan), and beyond — to learn from each other across borders.

🔧

Hack Together

No prizes, no competition — just collaborative work on civic projects that matter. Participants set the agenda, not the organisers.

🌏

Build Regionally

Across the Asia-Pacific, our challenges are entangled — through shared histories, colonial legacies, and geopolitical dependencies that no single community can address alone. FtO is where those entanglements become conversations.
