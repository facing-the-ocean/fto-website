Code of Conduct

FtO is committed to a safe, inclusive, and welcoming environment for all participants.

⚠️ Draft — pending approval. The FtO Code of Conduct is under active development. See the working document (v0.1.1) and the Ground Rules Workshop slides from FtO Yokohama 2024.

The Facing the Ocean network is in the process of developing its own Ground Rules / Code of Conduct. You can follow the current version and discussions here:

  • FtO CoC Version 0.1.1
  • FtO Ground Rules Workshop (FtO Yokohama 2024)
  • Community Survey — v0.1.1

While the FtO CoC is Under Development

Please refer to the Code of Conduct of each organising community. Before participating in an FtO event, carefully read and follow the Code of Conduct of the host community.

For example, when attending an FtO event in Taiwan, please read and follow the g0v Code of Conduct.
